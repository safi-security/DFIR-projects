# DFIR-projects

## Digital Forensics Investigation
Investigated an E01 forensic disk image using X-Ways Forensics. The investigation involved analysing artefacts, examining file system metadata, identifying USB activity, and constructing timelines to understand system activity.

## Malware Analysis
Performed malware analysis and reverse engineering using Ghidra as part of a group project analysing the Emotet malware. The analysis focused on understanding the behaviour of the malware and identifying key functions.

## Network Security
Configured and secured network devices using Cisco Packet Tracer. This included implementing device and administrator passwords, configuring AAA server authentication, and setting up GRE tunnelling between network devices.

## Docker
Completed coursework using Docker to create and run containerised environments.
